# FastTrack+ (FreshTrack)

A comprehensive full-stack application for meal planning, tracking nutrition, and managing grocery items and waste.

## Structure

This repository contains both the frontend and backend applications for the FreshTrack platform.

- **Frontend**: A React application built with Vite (`/FreshTrack_Frontend`)
- **Backend**: A Node.js/Express application (`/FreshTrack_Backend`)

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd FreshTrack_Frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

### Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd FreshTrack_Backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```

## Features
- Dashboard tracking
- Meal planning & nutrition tracking
- Grocery management
- Waste reduction statistics
